### Pointers and Arrays

* Array name = pointer to element 0 that can't be changed
* Pointer arithmetic becomes useful
    * walking down array
    * knows how much to add to address by pointer type
    * `p+1` is memory address of 2nd element
* BE EXTREMELY CAREFUL!
